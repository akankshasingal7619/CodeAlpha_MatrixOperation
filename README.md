# Matrix Operations in C

## 📌 Project Overview
This project is a menu-driven Matrix Operations program developed in C language as part of the CodeAlpha Internship. It allows users to perform various matrix operations through a simple command-line interface.

## ✨ Features
- Matrix Addition
- Matrix Multiplication
- Matrix Transpose
- Menu-Driven Program
- User-Friendly Interface
- Input Validation for Menu Choices

## 🛠 Technologies Used
- C Programming Language
- GCC Compiler
- Visual Studio Code

## 📂 Project Structure
matrix_operations.c

## ▶️ How to Run
1. Open the project in Visual Studio Code.
2. Compile the source code using a C compiler.
3. Run the executable file.
4. Select the desired matrix operation from the menu.
5. Enter the required matrix values.
6. View the output on the console.
